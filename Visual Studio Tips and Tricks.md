# Shortcuts

- CTRL + Space to open intelisense
- CTRL + F to find
- CTRL + N to open a new file
- CTRL + A to select All
- CTRL + ALT + P to preview css code location from the HTML document
- CTRL + Shift + P to open command pallete
- CTRL + Shift + N to open a new VSC window
- CTRL + [ or ] while selecting an area or having the mouse cursor in the line of code will move it from left or right!
- Alt+ ↑ / ↓ Move line up/down
- Shift+Alt + ↓ / ↑ Copy line up/down
- CTRL + SHIFT + A  while having the code selected you can wrap a elemnt like div around it.
